### 对象和类型

##### 1.变量

python 变量命名规则：

- 变量名只能包含字母，数字和下划线，但首字符不能是数字；
- 变量名不能包含空格；
- 不能将 python 关键字和函数名作变量名；

建议：

- 变量名应既简短又具有描述性；
- 慎用小写字母 `l` 和 大写字母 `O`，前者容易与数字 1 混淆，后者容易与数字 0 混淆。

##### 2.基本类型

python 中有五种基本对象类型：

- 字符串（string / str）: 使用单引号或双引号括起来的一系列字符
- 整数（integer / int）: 
  - 十进制：25
  - 八进制：025
  - 十六进制：0x25
- 浮点数（float）: 如：1.48，...
- 布尔数（boolean / bool）: True or False
- 复数（complex）: 1+1j

##### 3.type

python 提供了 `type()` 函数，用来检测数据类型
